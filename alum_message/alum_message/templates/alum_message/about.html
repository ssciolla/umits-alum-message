About the project

This project was created as part of the application process for a full-time position in the Teaching and Learning group at University of Michigan Information and Technology Services.

I would like to acknowledge my indebtedness to my former instructor, Anthony Whyte, who provided extensive guidance, resources, and starter code during Fall 2018 as part of UMSI 664: Database Application Design. In particular, his assignments and documentation supplied jumping off points for using Bootstrap/CSS, django_filters, and crispy_forms. The resources he created are available in a public GitHub repository.

If I had additional time, I would have attempted to fully implement authentication/authorization, likely by extending the User model to encompass the Alum model (following one of the strategies identified in the Django documentation).
