Alum Message

This application helps to connect University of Michigan alumni by making it simple to direct message one or more other alumni using only uniqnames.

The following four views are accessible through the Navigation Bar above:

  • View Messages allows users to search messages for those sent to or received by an alum with a given uniqname.
  • Send Message allows users to craft messages from and to alumni whose uniqnames are already registered with the application.
  • New Alum allows users to create a new alum record that can then be included subsequently in a new message.
  • About provides some additional details about this project and gives some acknowledgements.

Code for the project can be found in a private GitHub repository.
